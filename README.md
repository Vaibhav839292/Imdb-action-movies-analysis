## 📌 Analytical Problems Solved (Recruitment-Ready)
*As a fresher, I designed this project to tackle real-world data challenges:*

* **Data Wrangling & Hygienic Cleaning:** Addressed messy real-world data by isolating missing values, identifying outliers, and flattening nested strings (like multiple genres/directors) into structured formats.
* **Feature Engineering:** Extracted hidden signals from raw dates into temporal components (years and decades) to map the evolution of action cinema.
* **Dynamic Business Insights:** Replaced static charts with fully interactive Plotly dashboards (Scatter, Histograms, and Bar Charts), enabling stakeholder-like filtering.
* **Exploratory Data Analysis (EDA):** Discovered the non-linear relationship between production budgets, marketing popularity, and final IMDb critical ratings.
