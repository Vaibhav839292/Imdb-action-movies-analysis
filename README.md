# 🎬 IMDb Action Movies Analysis & Visualization

This project analyzes IMDb Action Movies data using Python. It includes data cleaning, exploratory data analysis (EDA), feature engineering, and interactive visualizations created with Plotly.

## 🛠️ Tech Stack Used

- Python
- Pandas
- NumPy
- Plotly
- Google Colab

---

## 📌 Project Highlights

- Cleaned missing and inconsistent data.
- Converted date columns into proper datetime format.
- Extracted release year and decade for analysis.
- Performed Exploratory Data Analysis (EDA).
- Created interactive charts using Plotly.
- Analyzed ratings, popularity, vote counts, and genre distribution.
